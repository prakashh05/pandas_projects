# Toyota Used Car Analysis

## Project Overview
This project focuses on analysing a dataset of Toyota used cars to explore patterns in pricing, mileage, year, reviews, and overall market trends. The goal was to understand how different factors affect car prices and to practice data analysis skills using Python.

---

## Tools & Libraries Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## Purpose of This Project
The main purpose of this project was to:
- Practice using Pandas for data manipulation
- Improve data visualisation skills
- Understand real-world dataset analysis
- Perform exploratory data analysis (EDA)
- Build a beginner data analysis portfolio project
- Built basic insights using correlation analysis

---

## Dataset
The dataset contains Toyota used car listings, including:
- Name
- Price
- Mileage
- Review ratings
- Review count
- Year
- Model
- Badge
